# Adelaide Airport Weather Analysis

This project uses R to analyse rainfall and temperature data from Adelaide Airport. It includes rainfall visualisations, formatted tables, and temperature regression analysis.

## Files

| File | Description |
| --- | --- |
| `Jiaqi Liuannual_rainfall_plot_code.R` | Creates an annual rainfall trend plot |
| `Jiaqi Liu monthly_rainfall_boxplot_code.R` | Creates a monthly rainfall boxplot |
| `Wei Hong Monthly Rainfall Distribution at Adelaide Airport Over 30 Years code.R` | Creates a formatted monthly rainfall table using `gt` |
| `yuchun_li_code.R` | Summarises rainfall data and creates comparison plots |
| `cai yifan quarterly_temperature_regression_clean_labels.R` | Runs quarterly temperature regression analysis |
| `zhongchi wang a1975897.R` | Runs annual mean temperature linear regression analysis |

## Requirements

The project requires R and the following packages:

```r
install.packages(c("tidyverse", "readxl", "gt", "dplyr", "ggplot2", "tidyr"))
```
